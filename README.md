# base64-for-hack
a baes64coder but not the nomarl one
安全BASE64工具 - 使用说明
================================

程序功能：
- 基于自定义密钥的BASE64编码加密
- 图形化界面，易于使用
- 支持文本的加密和解密

加密算法：
- 基于标准BASE64的自定义变体
- 使用密钥进行字符表置换
- 增加异或运算增强安全性

注意事项：
- 请妥善保管您的密钥
- 本程序生成的加密数据需要相同的密钥才能解密
- 建议不要修改源代码中的密钥
